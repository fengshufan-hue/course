# University Smart Course Schedule Pro

智能大学课程表管理系统，支持课程管理、周次导航、任务提醒、日历导出等功能。

## 功能

- **课程表展示** — 按周次和节次显示课程安排
- **周次导航** — 支持跳转任意教学周
- **深色模式** — 支持亮色/深色主题切换
- **课程管理** — 添加、编辑、删除课程（管理员权限）
- **任务管理** — 为课程添加作业/考试提醒
- **数据导出** — 导出为 iCal (.ics) / CSV 格式
- **数据备份** — JSON 格式导入/导出
- **单双周支持** — 自动识别奇数周、偶数周课程

## 技术栈

- React 18
- Vite
- Tailwind CSS
- Zustand (状态管理)
- date-fns (日期处理)

## 本地运行

```bash
# 使用任意静态服务器
python -m http.server 8080
# 或
npx serve .
```
